1  二元正态分布的模拟

可以使用一些函数，生成服从多元正态分布的随机样本。这些函数中，MASS包的mvrnorm( )是比较基本的。mvtnorm包给出了函数rmvnorm( )。在这里，我们使用的是来自VGAM包的binorm( )，这是一组函数，其中rbinorm( )可以直接生成二元正态分布的随机样本（pbinorm( )计算概率，dbinorm( )给出概率密度）（图-1）：

library(tidyverse)library(VGAM)
set.seed(1234)binorm1<-rbinorm(n=2000,mean1 = 0, mean2 = 0, var1 = 1, var2 = 4, cov12 = 0)
 binorm1%>%   as_tibble()%>%   ggplot(aes(x=X1, y=X2)) +   geom_point(shape=21,color="grey50",fill="yellow") +   theme_bw()

图-1

在rbinorm( )这个函数中，参数var1和var2对应两个随机变量的方差。


2   协方差阵对二元正态分布的影响

二元正态分布随机向量协方差矩阵是对称阵，主对角线元素是两个随机变量的方差，副对角线元素是两个随机变量的协方差，协方差取值的大小和正负决定了样本的分布形态。下面一个例子，协方差分别取成-1.5、0和1.5，各自模拟二元正态分布的样本并绘制散点图（图-2）。因为这一个迭代任务，我们可以对模拟和绘图的过程使用purrr包的map( )函数取代显式的for循环：

set.seed(1234)
cov12<-c(-1.5,0,1.5)binorm_data<-purrr::map(cov12,rbinorm,n=1000,mean1 = 0, mean2 = 0, var1 = 1, var2 = 4)
binorm_scatter<-function(data){  p<-data%>%  as_tibble()%>%       ggplot(aes(x=X1, y=X2)) +      geom_point(shape=21,color="grey50",fill="yellow") +      theme_bw()  }  bsplot<-purrr::map(binorm_data,binorm_scatter)  
library(patchwork)bsplot[[1]]+bsplot[[2]]+bsplot[[3]]

图-2

purrr包的map( )函数实现了高效的向量化计算，可以对一个向量或者列表（可以是数据集也可以是函数参数）的每个分量执行相同的函数，返回一个列表。在这里，我们将绘图过程编写为一个函数，最后的拼图使用了patchwork包的方法。


3   椭圆等高线：二元正态分布的几何特征

从图-2可见，二元正态分布的样本呈现出一种椭圆性。观察二元正态分布密度函数的指数部分，给定一个正常数d，它构成了一个椭球面。

给定不同的d值，可以绘制出一组椭圆形的等高线。这些等高线对应着二元正态随机向量概率密度的不同取值。

我们通过等高线图（counter图）展示这种性质。下面介绍两种方法，一种是使用ggplot2的stat_ellipse( )函数，根据不同的密度值绘制椭圆等高线。这种方法适合分布样本存在的场景，我们把上面图-2的代码稍作修改（图-3）：

set.seed(1234)
cov12<-c(-1.5,0,1.5)binorm_data<-purrr::map(cov12,rbinorm,n=1000,mean1 = 0, mean2 = 0, var1 = 1, var2 = 4)
binorm_scatter<-function(data){  p<-data%>%  as_tibble()%>%       ggplot(aes(x=X1, y=X2)) +      geom_point(shape=21,color="grey50",fill="yellow") +      stat_ellipse(type="norm",level=0.2,linetype=2,color="red")+      stat_ellipse(type="norm",level=0.5,linetype=2,color="red")+      stat_ellipse(type="norm",level=0.7,linetype=2,color="red")+      stat_ellipse(type="norm",level=0.9,linetype=2,color="red")+      theme_bw()  }  bsplot<-purrr::map(binorm_data,binorm_scatter)  

图-3

另一种方法是使用ellipse包的ellipse()直接生成counter图（图-4）：

library(ellipse)
level<-seq(0.1:0.9,by=0.2)
ellipse_d<-function(level){  aa<-ellipse(-0.75, c(1,2), c(0,0), level, npoints = 1000+level*100)  ellipse_data<-tibble(x=aa[,1],y=aa[,2],lev=factor(rep(level,nrow(aa))))  }ellipse_data<-purrr::map(level,ellipse_d)%>%  bind_rows()  ellipse_data%>%ggplot() +    geom_point(aes(x = x, y = y,color=lev),size = 0.1) +    scale_colour_grey(start = 0.7, end = 0.3) +        ## Use gray scales instead of colored default ##    labs(color = "Contours",          title = "Contour Plot"           )+   theme_bw()

图-4

使用ellipse( )这个函数时有一点要注意，外圈的椭圆要多生成一些点。


4   2D密度图

我们也可以给定若干格点，通过二元正态分布的概率密度函数绘制更“精确”一些的2D密度图像。这样的图像可以用ggplot2的geom_raster( )或者geom_tile( )来绘制（图-5）：

mu1<-0 #mean of X_1mu2<-0 #mean of X_2sigma11<-1 #variance of X_1sigma22<-4 #variance of X_2sigma12<-0 #covariance of X_1 and X_2 
###密度函数z <- function(x1,x2){ z <- exp(-(sigma22*(x1-mu1)^2+sigma11*(x2-mu2)^2-2*sigma12*(x1-mu1)*(x2-mu2))/(2*(sigma11*sigma22-sigma12^2)))/(2*pi*sqrt(sigma11*sigma22-sigma12^2)) }
###建立格点binorm2<-tibble(X1=rep(seq(mu1-3, mu1+3, length= 100),each=100),  X2=rep(seq(mu2-3, mu2+3, length= 100),times=100))%>%  mutate(density=z(X1,X2))
binorm2%>%  ggplot(aes(x=X1, y=X2,z=density)) +  geom_raster(aes(fill = density)) +  scale_fill_viridis_c()+  geom_contour(color="grey50")+  theme_minimal()

图-5

使用plotly包还可以绘制交互的2D密度（图-6）：

library(plotly)
x1 <- seq(mu1-3, mu1+3, length= 100)x2 <- seq(mu2-3, mu2+3, length= 100)z <- function(x1,x2){ z <- exp(-(sigma22*(x1-mu1)^2+sigma11*(x2-mu2)^2-2*sigma12*(x1-mu1)*(x2-mu2))/(2*(sigma11*sigma22-sigma12^2)))/(2*pi*sqrt(sigma11*sigma22-sigma12^2)) }f <- t(outer(x1,x2,z))
plot_ly(x=x1,y=x2,z=f,type"contour")%>%layout(xaxis=list(title="x1"),yaxis=list(title="x2"))

图-6

6   边缘分布

二元正态分布有一个很重要的性质：一个服从二元正态分布的随机向量，它的边缘分布（也就是向量中两个随机变量的分布）是正态分布。使用ggExtra包的函数ggMarginal( )，将联合分布和边缘分布展示在一张图中（图-8）：

library(ggExtra)
p <-  binorm1%>%      as_tibble()%>%      ggplot(aes(x=X1, y=X2)) +      geom_point(shape=21,color="grey50",fill="yellow") +      theme_bw()
ggMarginal(p, type="density")

图-8

可见，边缘分布是一元正态分布。
